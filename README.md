# sammify
Sammify your picture! Based on [this idea](https://samwierema.nl/sam.html) by @samwierema, this tools allows you to upload a photo and sammify it

# requirements

* python

# installation

* `pip install -r requirements.txt`
* `manage.py backend`
* `manage.py server`
